# Resilient CO Live Coding

A small Python coding challenge for a mini Venmo-style application.

## Project Overview

This repository contains a lightweight payment app prototype with:

- user account creation
- balance and credit card payment handling
- social activity feed rendering
- friend relationships

## Structure

- `main.py` / `main_original.py` - exercise implementation and entry points
- `src/` - package source code
- `tests/` - unit tests
- `requirements-dev.txt` - development dependencies
- `pytest.ini` - pytest configuration

## Getting Started

1. Create and activate your Python environment.
2. Install development dependencies:

```bash
python -m pip install -r requirements-dev.txt
```

## Running Tests

```bash
pytest
```

## Notes

The exercise is designed to implement user payments, credit card validation, activity feeds, and friend management in a minimal Venmo-like system.
